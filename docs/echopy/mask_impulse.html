mask_in_ryan(Sv, r, m, thr)
Mask impulse noise following the two-sided comparison method described
in:        
    Ryan et al. (2015) ‘Reducing bias due to noise and attenuation in 
    open-ocean echo integration data’, ICES Journal of Marine Science,
    72: 2482–2493.
 
Before comparing pings, samples within each ping in the original echogram 
are linearly averaged to a specified vertical resolution (bins), to smooth 
out vertical sample-to-sample variation. Then bins are compared with either 
side pings. If a threshold is excedeed, the bin will be masked.
 
Example:        
     resample every     |  two-side comparison  | mask if exceed threshold
        3 samples       |   after resampling    | (e.g., =2) in both sides
    ____________________|_______________________|_________________________   
     ping:    new ping: |  ping-1  ping  ping+1 |  ping-1  ping  ping+1
      2           3     |     2     3     8     |     2     3     8
      6           3     |     2     3     8     |     2     3     8
      1           3     |     2     3     8     |     2     3     8
      4  ----->   4     |     1     4     7     |     1     x     7
      1           4     |     1     4     7     |     1     x     7
      7           4     |     1     4     7     |     1     x     7
      2           2     |     3     2     4     |     3     2     4
 
Args:
    Sv (float): 2D numpy array with Sv data to be masked (dB) 
    r (float):  1D numpy array with range data (m)
    m (int): bin vertical length (m)
    thr (int): user-defined threshold value (dB)
    
Returns:
    bool: 2D numpy array mask (impulse noise = True)
    float: 2D numpy array with Sv data masked with NAN values
